# Kontoaktivität & Info

Dieser Bereich bietet Transparenz über deine Nutzung und technische Details deines Kontos.

## 1. Kontoübersicht
Hier findest du deine eindeutige **Benutzer-ID (UUID)**. Diese ist besonders wichtig für die Konfiguration von Memory-Policies oder bei Support-Anfragen.

## 2. Sitzungsübersicht (Audit)
Ontheia protokolliert alle aktiven Web-Sitzungen:
- **Zeitpunkt:** Wann wurde die Sitzung gestartet.
- **Gerät (Agent):** Welcher Browser oder welche App wurde genutzt.
- **IP-Adresse:** Von welchem Standort aus erfolgte der Zugriff.

## 3. Letzte Aktivitäten
Eine Liste der letzten KI-Ausführungen (Runs). Du siehst sofort den Status (Erfolg/Fehler) sowie den genutzten Agenten und Task.

## 4. Session-Token (API-Zugriff)
Dein persönliches Token für den technischen Zugriff auf die Ontheia-API.
- **Zweck:** Ermöglicht die Automatisierung von Aufgaben außerhalb der WebUI.
- **Wichtig:** Behandle das Token wie ein Passwort. Wer dieses Token besitzt, kann in deinem Namen mit Agenten kommunizieren und auf dein Gedächtnis zugreifen.
