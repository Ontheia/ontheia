### Pro-Tipp: Token-Überwachung
Bewege die Maus über eine Nachricht des Agenten, um rechts neben den Icons die Token-Statistik zu sehen (z. B. `1.200 / 45 Tokens`). Die erste Zahl steht für die gesendeten Informationen (Kontext/Prompt), die zweite für die Antwortlänge.
