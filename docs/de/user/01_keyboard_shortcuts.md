# Ontheia Keyboard Shortcuts & Bedienung

Um die Effizienz bei der Arbeit mit Ontheia zu steigern, wurden verschiedene Tastaturkürzel (Shortcuts) implementiert. Diese ermöglichen eine schnelle Navigation und Steuerung des Systems, ohne die Maus benutzen zu müssen.

## ⌨️ Globale Shortcuts

Diese Kürzel funktionieren in der gesamten Anwendung, sobald das Fenster im Fokus ist.

| Shortcut | Aktion | Beschreibung |
| :--- | :--- | :--- |
| `Strg` + `I` (oder `Insert`) | **Neuer Chat** | Erstellt sofort eine neue Chat-Sitzung. |
| `Strg` + `Entf` | **Chat löschen** | Löscht den aktuell aktiven Chat (nach Bestätigung). |
| `Strg` + `←` (Links) | **Sidebar umschalten** | Öffnet oder schließt die primäre Navigationsleiste (links). |
| `Strg` + `→` (Rechts) | **Sekundäre Sidebar** | Öffnet oder schließt die Info-Sidebar (rechts). |
| `Alt` + `Bild ↑` | **Vorheriger Chat** | Wechselt zum zeitlich davor liegenden Chat im Verlauf. |
| `Alt` + `Bild ↓` | **Nächster Chat** | Wechselt zum zeitlich danach liegenden Chat im Verlauf. |
| `Esc` | **Eingabefokus** | Springt sofort in das Textfeld des Chat-Composers zurück. |

## 🖱️ Navigation & Scrolling

Innerhalb der Chat-Ansicht helfen diese Kürzel beim schnellen Bewegen durch lange Konversationen.

| Shortcut | Aktion | Beschreibung |
| :--- | :--- | :--- |
| `Alt` + `↑` (oder `Pos1`) | **An den Anfang** | Scrollt die Konversation sofort ganz nach oben. |
| `Alt` + `↓` (oder `Ende`) | **Ans Ende** | Scrollt die Konversation sofort ganz nach unten zum neuesten Beitrag. |

---

## 💡 Tipps zur Bedienung

### Automatischer Fokus
Beim Laden oder Wechseln eines Chats wird das Eingabefeld automatisch fokussiert. Wenn Sie während einer Antwort des Agenten scrollen, können Sie jederzeit mit `Esc` den Fokus zurückholen, um sofort weiterzuschreiben.

### Sichtbarkeit der Shortcuts
Die Shortcuts für die Sidebars werden auch als **Tooltip** angezeigt, wenn Sie mit der Maus über die entsprechenden Buttons in der Topbar fahren.

### Keine Konflikte
Die gewählten Kombinationen wurden so optimiert, dass sie gängige Browser-Shortcuts (wie `Strg` + `T` für neue Tabs) nicht blockieren, um ein reibungsloses Arbeiten im Webbrowser zu ermöglichen.
