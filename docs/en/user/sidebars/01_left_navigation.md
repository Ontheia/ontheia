# Left Sidebar: Navigation & History

The left sidebar is your central starting point in Ontheia. It serves to organize your conversations and projects.

## 1. Actions & Search
- **New Chat:** Starts a new, empty conversation immediately.
- **Search:** Search your entire history (titles and IDs) for keywords to quickly find old conversations.

## 2. Projects (Structuring)
Ontheia allows you to organize chats into projects:
- **Folder Structure:** Projects can be nested hierarchically.
- **Organization:** You can move chats into projects via drag-and-drop (or via the context menu).
- **Context:** Projects often serve as a common framework for memory namespaces (see Admin documentation).

## 3. History (Chronology)
Here you find all chats that are not assigned to a specific project. 
- **Context Menu:** Via the "three-dot menu," you can rename, delete, or assign chats to projects.

## 4. User Menu (Footer)
At the bottom, you will find your profile:
- **Avatar:** Your personal picture or initials.
- **Administration:** (Admins only) Direct link to the Admin Console.
- **Settings:** Access to your user settings.
- **Logout:** Safely ends the current session.
