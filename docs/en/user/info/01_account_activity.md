# Account Activity & Info

This area provides transparency about your usage and technical details of your account.

## 1. Account Overview
Here you will find your unique **User ID (UUID)**. This is particularly important for configuring memory policies or for support requests.

## 2. Session Overview (Audit)
Ontheia logs all active web sessions:
- **Timestamp:** When the session was started.
- **Device (Agent):** Which browser or app was used.
- **IP Address:** From which location the access took place.

## 3. Recent Activities
A list of the latest AI executions (Runs). You immediately see the status (Success/Error) as well as the agent and task used.

## 4. Session Token (API Access)
Your personal token for technical access to the Ontheia API.
- **Purpose:** Enables the automation of tasks outside the WebUI.
- **Important:** Treat the token like a password. Anyone who possesses this token can communicate with agents in your name and access your memory.
