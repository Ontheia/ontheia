# The Chat Area

The Chat area is the heart of Ontheia. Here you conduct dialogues with AI agents, start complex workflows, and monitor the execution of tools.

## Window Structure

1. **Top Bar:** 
    - **Sidebar Toggle:** Switches the left and right sidebars on or off to create more space for text.
    - **Breadcrumbs:** Shows you which project you are currently in.
2. **Message History:** The middle area shows the dialogue in chronological order. Here, text messages, file attachments, and tool interactions are visualized.
3. **Composer (Input Area):** At the bottom is the input field along with the selectors for agents and tasks.
