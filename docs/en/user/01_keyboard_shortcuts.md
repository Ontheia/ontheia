# Ontheia Keyboard Shortcuts & Navigation

Ontheia provides keyboard shortcuts for efficient navigation and control without using the mouse.

## ⌨️ Global Shortcuts

These shortcuts work throughout the entire application whenever the window has focus.

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Ctrl` + `Insert` | **New Chat** | Opens a new chat session immediately. |
| `Ctrl` + `Delete` | **Delete Chat** | Deletes the currently active chat (with confirmation). |
| `Ctrl` + `←` (Left) | **Toggle Sidebar** | Opens or closes the primary navigation sidebar (left). |
| `Ctrl` + `→` (Right) | **Secondary Sidebar** | Opens or closes the activity/info sidebar (right). |
| `Alt` + `Page Up` | **Previous Chat** | Switches to the chronologically earlier chat in the history. |
| `Alt` + `Page Down` | **Next Chat** | Switches to the chronologically later chat in the history. |
| `Esc` | **Focus Input** | Returns focus to the chat composer text field immediately. |

## 🖱️ Navigation & Scrolling

These shortcuts help navigate through long conversations in the chat view.

| Shortcut | Action | Description |
| :--- | :--- | :--- |
| `Alt` + `↑` (or `Home`) | **Scroll to Top** | Scrolls the conversation immediately to the very beginning. |
| `Alt` + `↓` (or `End`) | **Scroll to Bottom** | Scrolls the conversation immediately to the latest message. |

---

## 💡 Usage Tips

### Automatic Focus
When loading or switching chats, the input field is automatically focused. If you scroll during an agent response, you can always press `Esc` to return focus to the composer.

### Shortcut Tooltips
The sidebar shortcuts are also displayed as **tooltips** when hovering over the corresponding toggle buttons in the top bar.

### No Conflicts
These key combinations are designed to avoid conflicting with common browser shortcuts (such as `Ctrl` + `T` for new tabs), ensuring a smooth experience in the browser.
