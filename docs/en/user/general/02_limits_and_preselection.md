# Limits & Preselection

Optimize your workflows with individual default settings.

## 1. Sidebar History & Limits
Control how much history Ontheia provides in the left sidebar for quick access:
- **Messages (Chats):** Number of recent sub-conversations displayed.
- **Status Messages & Warnings:** Limit on entries in the activity lists.
- **Range:** 5 to 50 entries.

## 2. Default Picker (Start Configuration)
Define which combination is automatically selected when opening a new chat:
- **Preferred Provider/Agent:** Which identity do you start with most often?
- **Standard Model/Task:** Which model or specific task should be preselected?

## 3. Tool Authorization (Standard)
Determine the security behavior for tool calls:
- **Request Authorization:** Every access to external tools must be manually confirmed by you.
- **Full Access:** The agent may use tools without further inquiry (only recommended for trusted workflows).
- **Blocked:** Tools are fundamentally deactivated.
