# Admin Console › Info

**Path:** Avatar dropdown → Administration → Info

Read-only page with no editable fields. Shows four tiles:

**Environment Status** — Available providers/models (count) and current chain steps (count).

**Security & Compliance** — Bullet points on RBAC, secret injection, audit logs, and hardening profiles.

**Software Version** — Current WebUI/host version.

**Admin Session Token** — Displays the current authentication token (masked by default). Buttons: **[Show / Hide Token]** · **[Copy Token]**.
